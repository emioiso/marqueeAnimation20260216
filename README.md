# Marquee Animation – EMI OISO

## 概要

本プロジェクトは、EMI OISOポートフォリオアプリに組み込むための  
SwiftUIによる横スクロール型マーキーコンポーネントの実装練習です。

単なるテキストアニメーションではなく、  
「動きそのものをインターフェースとして扱う」ことを目的としています。

---

- SwiftUIで横方向に流れるテキスト表現を実装する
- 再利用可能なMarqueeコンポーネントを構築する

- コンセプト性（寿司テーマ）を保ちながらUI実験を行う

# Overview

This project is a practice implementation of a horizontal marquee animation component in SwiftUI,  
designed for integration into the EMI OISO portfolio application.

The goal is not only to animate text, but to explore motion itself as an interface element.

---

## Purpose

- Build a reusable SwiftUI marquee component
- Implement continuous horizontal text animation
- Prepare for link-based navigation integration
- Maintain a concept-driven (sushi-themed) UI experiment

---

## Technologies

- SwiftUI
- UIKit (UIFont for text width measurement)
- GeometryReader
- Linear animation for continuous scrolling

---

## Future Plans

- Integrate into EMI OISO portfolio app
- Expand into interactive link navigation
- Improve seamless infinite scrolling
- Develop motion-based interface concepts
